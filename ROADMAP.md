# 🗺️ Roadmap — AI Medicament Scanner

This document outlines the planned development path for the AI Medicament Scanner project.

## 🏁 Phase 1: Foundation (Current)
- [x] Initial UI/UX Design (Material 3)
- [x] Layered Architecture (lib/ subfolders)
- [x] MultiProvider State Management
- [x] Google ML Kit OCR integration stub
- [x] Local Medication Database (20+ entries)
- [x] Unified Results Screen

## 🚀 Phase 2: Intelligence (High Priority)
- [x] **Basic ML Kit Integration**: Text recognition structure established.
- [ ] **Medication Database Expansion**: Target 100+ common medications (Currently ~20).
- [ ] **AI Natural Language Summary**: Use Gemini/OpenAI API to explain lab results.
- [ ] **Multi-language Support**: Arabic, French, Spanish, and German localization.

## 📱 Phase 3: User Experience (Medium Priority)
- [x] **Dark Mode Support**: Full system-aware dark theme implemented.
- [x] **Local History**: Store past scans securely using `shared_preferences`.
- [ ] **PDF Export**: Generate shareable reports of analysis findings.
- [ ] **Profile Management**: Store basic user context (allergies) for personalized warnings.
