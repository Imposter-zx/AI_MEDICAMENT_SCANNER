# ⚡ Quick Start — AI Medicament Scanner

Get the project running on your development machine in under 5 minutes.

## 1. Prerequisites
- [Flutter SDK](https://flutter.dev/docs/get-started/install) (Stable channel)
- A physical device or emulator (Android or iOS)

## 2. Setup
```bash
# Clone the repository
git clone https://github.com/yourusername/ai_medicament_scanner.git
cd ai_medicament_scanner

# Install packages
flutter pub get
```

## 3. Run
```bash
# List available devices
flutter devices

# Run on your chosen device
flutter run
```

## 4. Key Features to Test
1. **Medication Scan**: Click "Medication Scan" -> Upload an image of an "Aspirin" or "Paracetamol" box to see the database match.
2. **Document Analysis**: Upload a photo of a lab report containing "Hemoglobin" to see it extract values and flag them if abnormal.
3. **Results**: Review the structured display and safety disclaimers.
